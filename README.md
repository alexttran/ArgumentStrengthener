# Argument Strengthener

A tool that pressure-tests opinions by generating steelman counterarguments, identifying weak points, and suggesting reinforcement strategies.

## Structure

- `/backend` - Node.js + Express API with OpenAI integration
- `/frontend` - React + Vite user interface

## Setup

1. Install dependencies in both folders: `npm install`
2. Copy `backend/.env.example` to `backend/.env` and add your OpenAI API key
3. Start backend: `cd backend && npm run dev`
4. Start frontend: `cd frontend && npm run dev`
